# ferreteria-app
# Sistema de Gestión de Ferretería

Proyecto web desarrollado como caso de estudio para la gestión básica de una ferretería.

##Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- MySQL
- Git / GitHub

##Funcionalidades
- Registro y gestión de productos
- Operaciones CRUD (crear, leer, actualizar y eliminar)
- Validación de formularios
- Conexión con base de datos MySQL

##Mi aporte
- Desarrollo del frontend de la aplicación
- Diseño e implementación de la base de datos
- Creación de consultas SQL
